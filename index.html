Tonal StarField

A massively collaborative musical composition simulation

Created by SIGMusic / @SIGMusicUIUC

How does this work?

Web page

Responsible for collecting user input. Communicates to the web server through WebSockets

HTML // CSS // Javascript // WebSockets

Web server

Relays data from users to visual simulation.

node.js // OSC

Visual Simulation

Creates a visual representation of physical and digital input. Plays sounds and controls lights through OSC

Unreal Engine // OSC

Sound

Uses ChucK to build sound synthesizers from the ground up.

ChucK // OSC

Lights

Uses custom-built lighting hardware and bluetooth protocol to light the room.

Bluetooth // Arduino // Voodoo Magic
