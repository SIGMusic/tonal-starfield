Tonal Starfield

A massively collaborative musical and visual experience

Created by SIGMusic // @SIGMusicUIUC

How does this work?

Web page

Responsible for collecting user input. Communicates to the web server through WebSockets

HTML // CSS // Javascript

Web server

Relays data from users to visual simulation.

node.js

Visual Simulation

Creates a visual representation of physical and digital input. Plays sounds and controls lights

three.js // cannon.js // WebSockets

Sound

Uses ChucK to build sound synthesizers from the ground up.

ChucK // OSC

Lights

Uses custom-built lighting hardware to light the room. Bluetooth used to send lighting commands wirelessly.

Bluetooth // Arduino // Magic

About Us

SIGMusic (Special Interest Group for Music) is a student-run programming group for algorithmic music composition, computer music synthesis, visual simulations and more.
